BGW - BioFabric Graph Workspace

Software for visualising clique and community motifs modelled using integer linear programming (ILP) in BioFabric visualization.

Graph

Force-directed visualization of the loaded JSON graph.

Pipeline

Create pipelines to generate ILP problems and run Gurobi on loaded graphs.

Dataset

Upload new graphs in JSON format or manage existing ones.

Generator

Generate clique graphs with custom parameters and save them as JSON.

Scheduler

View and manage scheduled and running pipelines.

Analysis

Analyze saved pipelines/queues and view the comparative clique chart.

Staircase Results

Load local JSON and SOL files and view the Biofabric layout with a staircase overlay.
